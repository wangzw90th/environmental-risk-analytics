# Environmental Risk Analytics

Analytical workflows for understanding evolving risk patterns in complex environmental and infrastructure systems.

This repository contains quantitative workflows and exploratory tools for:

- risk evolution analysis
- geospatial environmental data
- time-series modelling
- uncertainty quantification
- operational risk indicators
- monitoring and forecasting systems

Applications include:

- geothermal systems
- induced seismicity
- environmental monitoring
- climate-related risk analysis
- infrastructure resilience

## Current Topics

- Risk evolution and transition detection
- Geospatial migration analysis
- Operational risk indicators
- Time-series and uncertainty analysis
- Environmental system monitoring
