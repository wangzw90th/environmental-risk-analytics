# Subsurface Monitoring & Risk Analytics

Analytical workflows for understanding evolving subsurface behaviour, monitoring indicators, and risk evolution in geoenergy and natural hazard systems.

This repository contains exploratory workflows and visualizations focused on transforming geophysical observations into interpretable monitoring metrics and decision-oriented risk indicators.

## Core Topics

- subsurface monitoring
- induced seismicity assessment
- seismic hazard evaluation
- uncertainty quantification
- monitoring and forecasting systems
- operational risk indicators
- geophysical data analytics

## Application Areas

- geothermal reservoir monitoring
- induced seismicity
- geoenergy systems
- seismic hazard assessment
- natural hazard monitoring
- infrastructure resilience

---

## Motivation

Many environmental and geoenergy systems generate large volumes of observational data, yet decision-making often depends on a small number of interpretable indicators.

This repository explores how complex geophysical observations can be transformed into:

- monitoring indicators
- risk metrics
- uncertainty-aware forecasts
- decision-support information

The goal is to bridge geophysical observations, system dynamics, and operational decision-making in complex subsurface systems.
