# Case Study 01
## Operational Controls on Seismic Migration

### Question

Can operational variables explain the evolution of seismic migration fronts during geothermal stimulation?

### Study Sites

- Soultz-sous-Forêts, France
- Basel, Switzerland
- Utah FORGE, USA

### Data

- Seismic catalogs
- Injection records
- Wellhead pressure measurements
- Hydraulic energy estimates

### Methods

- Rolling maximum migration front
- Correlation analysis
- Site-specific regression analysis

### Outputs

- Monitoring indicators
- Operational forcing metrics
- Risk-relevant interpretations

### Workflow

Operational data
→ Migration-front extraction
→ Statistical analysis
→ Monitoring indicators
→ Decision support

### Key Insights

This workflow explores how operational forcing metrics can be linked to the evolution of seismic migration fronts during geothermal stimulation.

The framework is designed to support:

- monitoring-system development
- operational risk assessment
- cross-site comparison
- decision-oriented subsurface analytics

### Status

Ongoing research. Detailed results and quantitative findings will be presented in a future peer-reviewed publication.
